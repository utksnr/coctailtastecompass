# coctailtastecompass
A coctail taste compass radar chard visulization 
This is just a small side project I hacked together while I was working part-time as a bartender during my university years.

As a student, I was spending my days with statistics and my nights behind the bar. I thought it would be fun to combine the two and see if I could visualize the balance of a cocktail using data. It's not a professional-grade software, just a fun tool for people who like to see what's actually in their glass.

What it does
Radar Charts: It takes 200+ recipes and turns their taste profile (Sweet, Sour, Bitter, etc.) into a simple graph.

Basic Analytics: Just a quick way to see if a drink is too boozy or perfectly balanced.

Recipes: All the standard classic stuff and some modern ones I've gathered along the way.

Tech Stuff
I used Python because it's what we use in class, and Streamlit because I wanted to get the UI running as fast as possible without learning heavy web dev.

How to run
If you have Python installed, just do this:

pip install streamlit pandas plotly

streamlit run main.py
